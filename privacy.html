隐私说明

照片桌宠重视你的照片与个人数据。本说明介绍桌面软件如何处理数据。

本地数据

软件无需注册账号，不收集使用统计、设备标识或广告数据。用户导入的照片、生成的透明图片、角色设置和自定义台词均保存在用户电脑本地，不会上传至开发者服务器。

模型下载

正式安装包已内置抠图模型，可以离线处理照片。如果内置模型缺失或校验失败，软件可能从 Hugging Face 或其镜像下载固定版本的 IS-Net 模型。该请求会向下载服务暴露常规网络信息，例如 IP 地址和 User-Agent，但不会包含用户导入的照片。

卸载与删除

卸载软件不会自动删除位于用户数据目录中的照片和设置。如需彻底删除本地数据，请在退出软件后手动删除照片桌宠的用户数据目录。

网站访问

官网由 GitHub Pages 托管。访问网站或下载安装包时，托管服务可能根据其隐私政策记录必要的网络日志。

联系我们

当前可通过开发者的 GitHub 主页反馈隐私或产品问题。
