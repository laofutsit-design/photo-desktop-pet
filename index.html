下载安全 · 先核验，再安装

看到“通常不会下载”？

这是浏览器对未签名新软件的信誉提醒，不代表文件已被判定为病毒。请不要关闭 SmartScreen 或杀毒软件。

当前为未签名测试版

SmartScreen 尚未积累足够的发布者与文件信誉。校验值只能证明文件与本次 GitHub 发布一致，不能替代杀毒扫描。

  1. 01

    确认下载来源

    仅从本官网按钮或项目的 GitHub Release 下载。文件名出现“(1)”或“(2)”通常只是重复下载标记。

  2. 02

    核对 SHA-256

    在 PowerShell 运行 Get-FileHash .\photo-desktop-pet-1.0.8-Windows-x64.exe -Algorithm SHA256，结果应与下方完全一致。

    2D380FB9CCA2539D32DF0A75F71CDCD8A3B66FF66FCEEB1ED45914161388D1F7

    若校验值不一致，请立即删除文件，不要运行。

  3. 03

    确认无误后再保留

    在 Edge 下载面板中选择“…” →“保留”→“显示详细信息”→“仍然保留”。运行时若再次出现 SmartScreen，请先再次确认来源和校验值，再决定是否继续。
